# <p align='center'>Computandi</p>

<p align='center'>***Computandi*** is a basic calculator made on Python 3 with PySimpleGUI</p>

## Features
- Basic arithmetic operators including percent (%)
- Clear and equal buttons
- Fully-functioning GUI
